# House Price Prediction


### Project Overview
The objective was to build machine learning models that can predict house prices based on various property features and analyze the factors that influence pricing.

### Dataset

* Dataset: Housing Prices Dataset
* Source: Kaggle
* Target Variable: Price

### Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn
* Google Colab

### Tasks Completed

#### Task 1: Data Loading and Exploration

* Loaded the dataset using Pandas
* Explored rows, columns, and data types
* Checked for missing values

#### Task 2: Data Cleaning

* Removed duplicate records
* Converted categorical variables into numerical format
* Prepared the dataset for machine learning

#### Task 3: Model Building

Models Used:

1. Linear Regression
2. Random Forest Regressor

#### Task 4: Data Visualization

* House Price Distribution Histogram
* Correlation Heatmap
* Actual vs Predicted Price Scatter Plot

#### Task 5: Insights and Findings

* Area was one of the strongest factors affecting house prices.
* Number of bathrooms and stories also had a significant impact.
* Linear Regression performed better than Random Forest on this dataset.
* The model explained approximately 65% of the variation in house prices.
